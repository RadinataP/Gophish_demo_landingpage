# Portal KKN UIN IB – Demo Landing Page

Proyek ini merupakan implementasi halaman landing page untuk simulasi Portal KKN UIN Imam Bonjol Padang.
Dibuat sebagai bagian dari pembelajaran terkait keamanan informasi, khususnya pada studi kasus rekayasa sosial berbasis antarmuka web.

---

## 📌 Deskripsi

Landing page ini dirancang untuk meniru tampilan portal resmi dengan pendekatan UI modern dan responsif.
Fokus utama proyek ini adalah pada aspek tampilan, alur interaksi pengguna, serta simulasi skenario akses sistem.

---

## ⚙️ Teknologi yang Digunakan

* HTML5
* CSS3 (Glass effect & modern UI styling)
* JavaScript (interaksi dasar)
* GitHub Pages (deployment)

---

## 🚀 Cara Menjalankan

### 1. Clone Repository

```bash
git clone https://github.com/RadinataP/Gophish_demo_landingpage.git
```

### 2. Buka File

Buka file `index.html` menggunakan browser.

Atau gunakan Live Server (VS Code) untuk pengalaman pengembangan yang lebih optimal.
